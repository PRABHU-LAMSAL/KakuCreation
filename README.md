A modern, responsive website for KAKU CREATION photography and videography studio with an integrated shop feature.

- **Homepage** with hero section, services, gallery, and contact information
- **Shop Page** with product listings, shopping cart, and checkout functionality
- **Responsive Design** that works on all devices
- **Modern UI/UX** with smooth animations and transitions
- **Contact Form** for customer inquiries
- **Social Media Integration** with links to all platforms

## How to Use

1. Simply open `index.html` in a web browser
2. Navigate through the website using the menu
3. Visit the Shop page to browse and purchase products
4. Use the contact form to send inquiries
5. All cart data is saved in browser's local storage

## Files Structure

- `index.html` - Main homepage
- `shop.html` - Shop page with products
- `styles.css` - All styling
- `script.js` - Main JavaScript for homepage
- `shop.js` - Shop functionality and cart management
